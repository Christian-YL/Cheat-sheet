HTML Fundamentals

Skill Description Code example
<video> element The <video> element embeds a media player for video playback.
<img> element HTML image <img> elements embed images in documents. The src attribute contains the image URL and is mandatory. The <img> is an empty element meaning it should not have a closing tag.
<title> element The <title> element contains the text that defines the title of an HTML document. The title is displayed on the browser's title bar or tab in which the HTML page is displayed. The <title> element can only be contained inside a document's <head> element.
<aside> element The <aside> element is used to mark aditional information that can enhance another element, but isn'r required in order to understand the main content. Usually this information would be in a sidebar or a location where it doesn't obstruct the main piece of the content.
An example of this would be an article that discusses how to take care of a dog and next to the article an ad would appear advertising a dog grooming product.
<figure> and <figcaption> element(s) The <figure> element is used to encapsulate media such as an image, diagram, or code snippet. The <figcaption> element is used to describe the media encapsulated within the <figure> element.
Developers will normally use <figcaption> within the <figure> element to group the media and description; this way if a developer decides to change the position of the media, the description will go along with it.
Code <!--comment--> In HTML, comments can be added between an opening <!-- and closing -->. Content inside of comments will not be rendered by browsers, and are usually used to describe a part of code or provide other details.
Comments can span single or multiple lines.
